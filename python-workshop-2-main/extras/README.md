# Workshop 2 Extras

Advanced topics for self-study beyond the main workshop.

## Overview

This folder contains advanced Object-Oriented Programming concepts that build on what you learned in the main workshop. These topics are optional but highly valuable for writing more sophisticated Python code.

## Topics

### 01. Inheritance & Polymorphism

Learn how to create class hierarchies and write flexible, reusable code.

**Key Concepts:**

- Inheriting from parent classes
- Using `super()` to call parent methods
- Method overriding
- Polymorphism and common interfaces

[Explore Inheritance →](01_inheritance/)

### 02. Properties & Decorators

Master Python's property decorators for elegant attribute management.

**Key Concepts:**

- Problems with direct attribute access
- Getters and setters for validation
- The `@property` decorator
- Computed and read-only properties

[Explore Properties →](02_properties_decorators/)

## Learning Path

We recommend studying these topics in order:

1. **Start with Inheritance** - Build on your knowledge of classes
2. **Then Properties** - Learn modern Python patterns for attribute access

## When to Study These

- **After Workshop 2**: Make sure you're comfortable with basic OOP first
- **When you need them**: Learn inheritance when you find yourself duplicating code
- **For better code quality**: Properties make your code more Pythonic and maintainable

## Tips for Self-Study

1. Run each example file and study the output
2. Modify the code to experiment
3. Try creating your own examples
4. Don't rush - these are advanced concepts
